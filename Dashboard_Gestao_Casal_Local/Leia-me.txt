=====================================================
    GUIA DE INSTALAÇÃO - DASHBOARD FINANCEIRO
=====================================================

Para rodar este projeto em uma nova máquina, instale:

1. .NET 8 SDK:
   O motor que roda o C#. Baixe em: https://dotnet.microsoft.com/download/dotnet/8.0

2. SQLITE:
   Geralmente já vem no Windows, mas você pode baixar as "Tools" se precisar abrir o banco fora da API: https://www.sqlite.org/download.html

3. FERRAMENTA DE BANCO (Opcional):
   DB Browser for SQLite (para visualizar a tabela manualmente): https://sqlitebrowser.org/

4. EF CORE TOOLS:
   Abra o PowerShell e rode este comando para poder gerenciar o banco:
   dotnet tool install --global dotnet-ef

5. WHATSAPP (TWILIO):
   Lembre-se de que a nova máquina precisa de internet e os números devem estar 
   com o "join thick-laugh" ativo no Sandbox da Twilio.

CONFIGURAÇÃO INICIAL:
- Verifique se o arquivo 'appsettings.json' na API está com as chaves da Twilio corretas.
- Verifique se o endereço da API no 'Program.cs' do Blazor bate com a URL que a API vai rodar.
