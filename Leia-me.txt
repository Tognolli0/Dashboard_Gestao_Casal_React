GUIA RAPIDO DE INSTALACAO

1. Instale o SDK do .NET 8.
2. Instale o Node.js para o frontend.
3. Se precisar trabalhar com banco local, use SQLite apenas fora do Git.
4. Para migrations, instale `dotnet-ef`.
5. Configure Twilio, Azure e banco apenas com segredos locais.

CONFIGURACAO INICIAL
- Revise `MinhaVidaAPI/appsettings.Development.json` ou User Secrets.
- Nao salve senhas ou tokens em arquivos versionados.
- Verifique a URL da API usada pelo frontend antes de publicar.
